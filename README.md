# swe-691-lab1: AI-Assisted Debugging

Repository นี้จัดทำขึ้นเพื่อจัดเก็บงานในวิชา Software Engineering (รหัสวิชา 691) โดยเน้นไปที่การทดสอบทักษะการตรวจสอบและแก้ไข Bug ในโค้ด Python โดยใช้ความสามารถของ AI

## 📂 โครงสร้างไฟล์ใน Repository
- `grade-lab.py`: ไฟล์โปรแกรม Python สำหรับคำนวณเกรด (เวอร์ชันที่ได้รับการแก้ไขแล้ว)
- `ai-bugfix.md`: บันทึกรายละเอียดของ Bug ที่พบและการอธิบายแนวทางการแก้ไขโดย AI
- `Software Engineering 67332310030-5.pdf`: เอกสารรายงานประจำตัวนักศึกษา
- `README.md`: รายละเอียดภาพรวมของโปรเจกต์

## 🛠 การทดลอง: การหา Bug ใน Python
ในโปรเจกต์นี้ เราได้ทดสอบฟังก์ชัน `calculate_grade` ซึ่งมีปัญหาหลัก 2 ส่วนคือ:
1. **Indentation Error:** การย่อหน้าในภาษา Python ไม่ถูกต้องทำให้โปรแกรมไม่ทำงาน
2. **Zero Division Error:** ขาดการตรวจสอบกรณี List คะแนนว่างเปล่า ซึ่งจะทำให้โปรแกรม Error เมื่อทำการหาร

## 📝 สรุปสิ่งที่ได้เรียนรู้ (Trust Level)
จากการทดลองใช้ AI ในการช่วยหา Bug ผลสรุปที่ได้คือ:

> **"Trust Level 4: ควรเชื่อมั่นในระดับ 4 เนื่องจาก AI มีความแม่นยำสูงมากในการตรวจจับ Syntax และ Logic พื้นฐาน แต่ยังคงต้องใช้มนุษย์ตรวจสอบความถูกต้องขั้นสุดท้ายก่อนนำไปใช้งานจริง"**
